# Hesap Makinesi

Karanlık temalı, masaüstü kullanımına uygun basit bir **Hesap Makinesi** uygulaması.

> Bu repo, temel aritmetik işlemleri hızlıca yapabilmeniz için hazırlanmıştır.

---

## Özellikler

- Temel işlemler: **Toplama (+)**, **Çıkarma (-)**, **Çarpma (*)**
- **Yüzde (%)** hesabı
- **Temizleme (C)** ile ekranı sıfırlama
- **Sonuç (=)** ile işlem tamamlama
- Üst ekranda **işlem ifadesi / geçmiş**, ana ekranda **sonuç** gösterimi
- Modern **dark UI** ve büyük butonlar


## Kurulum / Çalıştırma

Aşağıdaki adımlar, projeyi GitHub üzerinden indirip yerelde çalıştırmak için genel akıştır (projenin dili/IDE’si fark etmeksizin).

1. Repoyu klonlayın:
   ```bash
   git clone <REPO_URL>
   cd <REPO_KLASORU>
   ```

2. Projeyi IDE ile açın (Visual Studio / VS Code / IntelliJ vb.)

3. Bağımlılık/derleme adımlarını çalıştırın (projeye göre değişir) ve uygulamayı başlatın.

---

## Kullanım

- Sayıları girin
- İşlem seçin (**+ / - / ***)
- Gerekirse yüzde (**%**) uygulayın
- Sonucu görmek için **=**’e basın
- Sıfırlamak için **C**’ye basın
