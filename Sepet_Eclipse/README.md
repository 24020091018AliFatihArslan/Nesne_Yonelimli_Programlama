# Sepet (Eclipse)

Bu proje, "Sepet" örnek uygulamasının **Eclipse IDE** ile sorunsuz açılıp çalıştırılabilecek şekilde düzenlenmiş halidir.

## Eclipse'e import
1. Eclipse → **File → Import...**
2. **Existing Projects into Workspace**
3. **Select root directory** → bu projenin klasörünü seç
4. **Finish**

## Çalıştırma
- `src/tr/edu/istiklal/ui/SepetForm.java` içindeki `main` metodu çalıştırılır.

## Paket yapısı
- Model sınıfları: `tr.edu.istiklal`
- Arayüz (Swing): `tr.edu.istiklal.ui`

## Notlar
- Toplam hesaplaması, ürünlerin `kdvHesapla()` metodu üzerinden yapılır.
- Proje Java 8 uyumludur.
